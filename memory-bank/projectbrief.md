# Project Brief: High Quality Info

## 核心目标
构建一个高质量的信息聚合与分析平台。通过集成多种数据源（如 RSS），利用 LLM（大语言模型）进行自动化摘要、翻译和分类，帮助用户高效获取高价值信息。

## 核心功能
1.  **多源信息聚合**: 支持 RSS 等动态数据源的订阅与抓取。
2.  **AI 增强处理**:
    - **双语摘要**: 自动生成中英文摘要。
    - **智能标签**: 自动提取内容标签。
    - **标题翻译**: 自动翻译外文标题。
3.  **多用户系统**:
    - **身份认证**: 支持 Email/Password, Google, GitHub 登录 (NextAuth)。
    - **个性化订阅**: 用户可独立管理订阅源 (UserSource)。
    - **状态隔离**: 独立的已读/收藏状态 (UserSignal)。
4.  **交互式看板**:
    - 基于卡片的信息展示。
    - 支持按标签、来源过滤。
    - 交互式详情查看。
